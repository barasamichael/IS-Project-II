Export Guidelines & Information

Data Export Information

Maximum Export Size: 50MB per export
Data Retention: Exports available for 30 days
Export Frequency: Maximum 10 exports per day
Processing Time: 1-5 minutes for most exports
Data Privacy: User consent respected for all exports

Data Security & Privacy

  • All exported data is encrypted during transmission
  • Personal user information requires explicit consent selection
  • Settlement insights are anonymized by default
  • Export files are automatically deleted after 30 days
  • Access to exports is logged for audit purposes

Export Format Guidelines

CSV Format: Best for data analysis and spreadsheet applications. Includes headers and is compatible with Excel, Google Sheets, and statistical software.
Excel Format: Ideal for detailed reporting with multiple sheets and formatting. Includes charts and summaries where applicable.
JSON Format: Perfect for technical integration and API consumption. Structured data format for developers and system integration.
